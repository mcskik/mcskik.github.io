Menu Magic

Like the dynamic fly-out menu on this website? As well as mouse activation, it responds to the arrow keys plus enter. It is easy to maintain as its contents are read from an XML file. It can work in four different orientations.

  • Across and down like this website.
  • Across and up like the Windows start menu.
  • Down and across from the left hand side.
  • Down and across from the right hand side.

Safety Net

Parental control internet filter. It is a Windows service written in C# .Net. Supplied with a pre-populated filter criteria database. This is a password protected Microsoft Access database which can be edited to add or amend the filter criteria.

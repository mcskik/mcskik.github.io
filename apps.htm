Mobile Apps

CashRadar

CashRadar Android app to allow users to keep track of all their spending and savings.

Lifeline

Lifeline Android app to allow users to keep track of weight, diet, and exercise metrics.

ShoppingRadar

ShoppingRadar Android app to allow users to create and use shopping lists for any shops they regularly visit.

TaskRadar

TaskRadar Android app to allow users to create TODO lists of tasks to be done or checklists to be ticked off. Tasks are broken down into a hierarchy of collections, groups, and tasks. This allows you to plan all the one-off or periodic tasks you need to do. Periodic tasks can be reset back to the beginning so that they can be reused repeatedly.

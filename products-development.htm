Editorial

An editing tool for use during software development. Very useful for generating code from a list of known member variables. Generates property accessor methods from a list of pre-written member variables. Can swap variables to the other side of the equals signs to reverse the assignments. This tool is written in C# .Net.

Journalist

A custom logging class which provides a standard method of logging for all applications. This tool is written in C# .Net.

Motive8

A code promotion and file synchronization tool which can promote code between development and production directories or keep two or more directory trees completely synchronized. It keeps separate copies of all versions of each file managed by the system thereby acting as a lightweight configuration management system. It has a back-out mode which will reverse the effect of a promotion script thereby enabling a quick and easy back-out of code in the event of problems with a live implementation for instance. The synchronization tool is very useful for efficient backups as only the changes are copied. This tool is written in C# .Net.

Search8

An advanced search tool which allows search arguments with sophisticated Boolean logic (AND|OR|NOT) together with file type filtering and regular expressions. This tool is written in C# .Net.

Valid8

A tool which validates XML files against the supplied XSD schemas. This tool is written in C# .Net.
